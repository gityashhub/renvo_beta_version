   - unnecessary centered layouts

3. Layout should be:

-------------------------------------------------
| Sidebar |        Main Content (100%)          |
|         |                                     |
|         |                                     |
|         |                                     |
-------------------------------------------------

There should NOT be a large empty area on the right.

4. The main wrapper should use layouts like:
   - display: flex
   - flex: 1
   - width: 100%
   - min-width: 0
   - overflow-x: auto only when necessary

   - Dashboard

7. Tables should use the available width.

8. Charts should stretch to available width.

9. Forms should expand naturally.

10. The layout should remain fully responsive:
    - Desktop uses full width.
    - Tablet adjusts correctly.
    - Mobile remains unchanged.

11. Search the entire project for components or wrappers using classes such as:
    - container
    - max-w-*
    - max-width

12. Do NOT change any functionality or business logic.
Only fix the layout so the application fully utilizes the available horizontal space.
